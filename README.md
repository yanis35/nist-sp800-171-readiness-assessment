# NIST SP 800-171 Readiness Assessment

**Client:** Confidential  
**Role:** GRC Analyst — Security Assessment Lead  
**Framework:** NIST SP 800-171 (Rev 2)  
**Compliance Score:** 53% (107 controls assessed)  
**Duration:** 6 weeks  

---

## Project Overview

A comprehensive readiness assessment against NIST SP 800-171 for a client seeking compliance with DFARS clause 252.204-7012. The assessment evaluated 14 control families across 107 security requirements, producing 18 deliverables spanning the full audit lifecycle — from scoping to executive presentation.
